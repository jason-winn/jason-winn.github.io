Airbnb Map Pin Clustering

Analyzing the Globe In Nanoseconds

Map pin clustering involves the algorithmic grouping of pins on a map. This project was my own from the outset until completion. It involved creating a prototype server and client application. Airbnb has millions of lines of listing data from which coordinates of a location may be extracted, and so performance and scalability were of the highest concern.

The Solution

I pre-processed the data using a modified quad-tree algorithm, represented the tree using a hex geohash, and assembled the resulting hashes into a trie for low time-complexity access. Client side re-clustering was performed using a modified K-means algorithm.
