Airbnb Messaging UI

Creating a Smooth Messaging Experience

Airbnbâ€™s messaging is one of the most-used components of the app, yet it was not performing up to standards, and received many user complaints in the app store. This project was my own from the outset until completion. Smooth scrolling, particularly on older devices, was a high priority for me. In addition, I wanted a fluid message input, failure states for messages, saved message templates, and faster load times for long message threads.

The Solution

I implemented paginated loading of threads on the client alongside a more intuitive caching policy. For smooth scrolling I used CoreGraphics draw code to create chat bubbles and converted these into stretchable Bitmaps, which could be rendered extremely fast by older devices. Furthermore, I reduced layer blending wherever possible. Loading times were reduced by profiling the layout subviews routine and optimizing code for pre-determining cell sizes in the collection view.
