Game Splash Screens

Let the Games Begin!

Map pin clustering involved the algorithmic grouping of Airbnb's millions of pins on a world map. This project was my own from the outset until completion. Performance and scalability were a top priority.

I've done a lot of work with games between my personal projects and my time at Zynga. I find that splash screens are a great way to let my design thoughts run free, unrestricted by implementation and engineering limitations.
